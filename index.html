About WeatherWaypoint

WeatherWaypoint is a lightweight weather exploration tool that lets users search cities or click locations on the map to instantly view real-time weather conditions. It demonstrates modern front-end development techniques using JavaScript, APIs, and interactive maps.

How to Use

• Enter a city in the search bar and click Check Weather.
• Click markers on the interactive map to view weather for major cities.
• The app displays temperature, humidity, and weather conditions in real-time.

Technology Used

• OpenWeatherMap API for real-time weather data
• Leaflet.js for interactive maps
• HTML, CSS, and JavaScript for the interface
• OpenStreetMap map tiles
