About WeatherWaypoint

WeatherWaypoint lets users explore live weather conditions by clicking cities on the map or searching by name. It uses modern front-end techniques with APIs and interactive maps.

How to Use

• Type a city and click Check Weather.
• Click markers on the map to see weather info.
• Real-time temperature, humidity, and conditions appear instantly.

Technology Used

• OpenWeatherMap API for weather
• Leaflet.js for interactive maps
• HTML, CSS, JavaScript
• OpenStreetMap tiles
