About WeatherWaypoint

Explore live weather worldwide. Click cities or search by name to see temperature, humidity, and conditions. Built with modern JS, Leaflet, and OpenWeatherMap API.

How to Use

• Type city & click Check Weather
• Click map markers for instant weather
• Toggle °F / °C (future feature)
• Mobile-friendly & responsive

Technology Used

• OpenWeatherMap API
• Leaflet.js for interactive maps
• HTML / CSS / JavaScript
• OpenStreetMap tiles
